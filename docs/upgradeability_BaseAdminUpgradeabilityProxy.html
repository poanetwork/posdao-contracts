    BaseAdminUpgradeabilityProxy

    contract BaseAdminUpgradeabilityProxy

    is UpgradeabilityAdmin, BaseUpgradeabilityProxy

    This contract combines an upgradeability proxy with an authorization mechanism for administrative tasks.

    • upgradeToAndCall

      function upgradeToAndCall(address newImplementation, bytes data) external payable

      Upgrade the backing implementation of the proxy and call a function on the new implementation. This is useful to initialize the proxied contract.

      Modifiers:
      ifAdmin
      Parameters:
      newImplementation - Address of the new implementation.
      data - Data to send as msg.data in the low level call. It should include the signature and the parameters of the function to be called, as described in https://solidity.readthedocs.io/en/v0.4.24/abi-spec.html#function-selector-and-argument-encoding.
